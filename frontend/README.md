# Frontend (React + TypeScript + Vite)

Run frontend (from the `frontend` folder):

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

The dev server runs at `http://localhost:5173` by default and calls the backend at `http://localhost:8000/todos`.

Structure:

- src/components: small reusable UI components
- src/pages: main pages (Todos page)
- src/services: Axios API wrapper
- src/hooks: data-fetching hook
- src/types: TypeScript types

This is a minimal Vite + React + TypeScript app intended for educational purposes.
